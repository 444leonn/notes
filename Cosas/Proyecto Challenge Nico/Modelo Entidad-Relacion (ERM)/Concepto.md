# Modelo Entidad Relacion

El modelo entidad-relacion o diagrama entidad-relacion, es una herramienta para el diseño conceptual de bases de datos que representa las entidades, atributos y relaciones dentro de un sistema de información. Es una representación visual de cómo los datos se organizan y se relacionan entre sí

## **Elementos clave del Modelo ER:**

### **Entidades:**

Representan objetos, personas, conceptos o lugares del mundo real sobre los que se quiere almacenar información. Por ejemplo, en una base de datos de una universidad, las entidades podrían ser "Estudiante", "Profesor", "Curso", etc.

### **Atributos:**

Describen las características o propiedades de las entidades. Por ejemplo, para la entidad "Estudiante", los atributos podrían ser "Nombre", "Apellido", "Número de matrícula", etc.

### **Relaciones:**

Indican cómo interactúan las entidades entre sí. Por ejemplo, la relación entre la entidad "Estudiante" y "Curso" podría ser "se matricula en".
