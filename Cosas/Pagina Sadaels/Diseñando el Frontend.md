# Estructura del sitio web

## **Página principal**: 

Dashboard con resumen de ventas, clientes destacados y productos con bajo stock

### Descripción detallada del Dashboard Principal

**Cabecera:**

- Logo de Sadaels en la esquina superior izquierda
- Barra de navegación con enlaces a: Dashboard, Clientes, Productos, Ventas
- Botón de perfil/configuración en la esquina superior derecha

**Panel principal (dividido en 3 secciones):**

- Sección 1: Resumen de ventas
    - Gráfico circular mostrando ventas por categoría de producto
    - Gráfico de barras mostrando ventas de los últimos 7 días
    - Indicador de ventas totales del mes con comparación al mes anterior
- Sección 2: Clientes destacados
    - Tarjetas con los 3-5 clientes que más compras realizaron este mes
    - Cada tarjeta muestra: foto/avatar, nombre, total gastado
- Sección 3: Productos con bajo stock
    - Tabla con productos que necesitan reposición
    - Columnas: Imagen, Nombre, Color, Talle, Stock actual
    - Indicador visual (color rojo) para items críticos

Para crear este mockup visualmente, te recomendaría usar herramientas como Figma, Adobe XD o incluso Canva que tienen plantillas para dashboards que podrías adaptar según esta estructura.

## **Sección de Clientes**:

- Buscador de clientes por nombre
- Ficha de cliente con datos personales (nombre, apellido, CUIT, DNI, dirección, teléfono, email)
- Historial de compras del cliente

## **Sección de Productos/Stock**:

- Listado de productos con filtros
- Formulario para añadir/modificar productos (nombre, imagen, descripción, color, talles, precio, cantidad)

## **Sección de Ventas**:

- Formulario de nueva venta
- Historial de ventas
- Sistema que actualice automáticamente el stock

# Paleta de Colores

## Modo Claro

### **Colores primarios**

- Azul principal: #3B82F6 (un azul vibrante pero profesional)
- Acento: #10B981 (verde para indicadores positivos/éxito)

### **Colores neutrales**

- Fondo: #FFFFFF (blanco)
- Fondo secundario: #F3F4F6 (gris muy claro)
- Texto principal: #1F2937 (casi negro)
- Texto secundario: #6B7280 (gris medio)

### **Colores de estado**

- Éxito: #10B981 (verde)
- Advertencia: #FBBF24 (amarillo)
- Error/Alerta: #EF4444 (rojo) - útil para los indicadores de bajo stock

## Modo Oscuro

### **Colores primarios**

- Azul principal: #60A5FA (un azul más claro que en el modo día)
- Acento: #34D399 (verde más claro para mejor contraste)

### **Colores neutrales**

- Fondo: #111827 (casi negro)
- Fondo secundario: #1F2937 (gris muy oscuro)
- Texto principal: #F9FAFB (blanco ligeramente apagado)
- Texto secundario: #9CA3AF (gris claro)

### **Colores de estado**

- Éxito: #34D399 (verde más claro)
- Advertencia: #FBBF24 (amarillo, mismo que en modo claro)
- Error/Alerta: #F87171 (rojo más claro)
